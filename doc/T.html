(static) pvalue(t, df, onetailopt) → {Number}

Calculate a two-tailed p-value or an one-tailed p-value of a T-test statistic and a degree of freedom which can obtain from module:statistic.T.statistic. It use a betaic or a cdf function internally.
Parameters:
Name Type Attributes Description
t Number A T-test statistic value
df Number A degree of freedom (ν) parameter of T distribution
onetail Boolean <optional>
 If it is true, return p-value of one-tailed test, otherwise it return p-value of two-tailed test. The default value is false.

(static) statistic(u1, s1, n1, u2, s2, n2) → {Array}

Return a test statistic of T-test which follow Student's T distribution. It use a Student's t-test when the variances of two samples are equal and a Welch's t-test otherwise. The equalness of the variances will be tested via F-test with 0.05 as a critical value.
Parameters:
Name Type Description
u1 Number A mean value (X1) of sample 1.
s1 Number A variance (s12) of sample 1.
n1 Number The number of samples (N1) of sample 1.
u2 Number A mean value (X2) of sample 2.
s2 Number A variance (s22) of sample 2.
n2 Number The number of samples (N2) of sample 2.

(static) student(u1, s1, n1, u2, s2, n2) → {Array}

Return a test statistic of T-test which follow Student's T distribution via a Student's t-test. It is used when the variances of two samples are equal. You may need to test if the two variances are equal or not via a F-test. Use a Welch's t-test instead otherwise.
