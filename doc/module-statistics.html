(static) beta(a, b, xopt) → {Number}

Calculate an approximation of a beta function or an incomplete beta function through a gamma function or a regularized incomplete beta function respectively.

If the third argument is equal to 1 or not specified, it return an approximation value of a beta function, otherwise it return an approximation value of an incomplete beta function.

(static) contfrac(n, fn) → {Number}

Calculate a continued fraction via a/b generator function. Ported from a python code in Rosetta Code

Parameters:
Name Type Description
n Number

The number of iteration
fn function

A function which obtain current index and return a list which contains values of a and b
Source:
  • statistics.js, line 106
See:
  • http://rosettacode.org/wiki/Continued_fraction#Fast_iterative_version
Returns:
Type
Number
Example
// return a value of sqrt(2)
var fsqrt2 = function(n) {
    return [n > 0 ? 2 : 1, 1];
};
statistics.contfrac(200, fsqrt2);

// return a value of e (napier)
var fnapier = function(n) {
    return [n > 0 ? n : 2, n > 1 ? n-1 : 1];
};
statistics.contfrac(200, fnapier);

// return a value of pi
var fpi = function(n) {
    return [n > 0 ? 6 : 3, Math.pow(2 * n - 1, 2)];
};
statistics.contfrac(200, fpi);

(static) range(start, stopopt, stepopt, digitsopt) → {Array}

Create an array with continueous numbers. If the step argument is omitted, it defaults to 1. If the stop argument is omitted, the value of stop is set to start and the value of start become 0. If the step argument is 0, it raise Error.

For a positive step, the contents of a range r are determined by the formula r[i] = start + step * i where i >= 0 and r[i] < stop.

For a negative step, the contents of the range are still determined by the formula r[i] = start + step * i, but the constraints are i >= 0 and r[i] > stop.

If digits argument is specified, it call toFixed method with the value before adding to the contents.

Parameters:
Name Type Attributes Description
start Number

A start or stop value.
stop Number <optional>

A stop value
step Number <optional>

A step value. Default is 1.
digits Number <optional>

A digits argument of toFixed method if specified
Source:
