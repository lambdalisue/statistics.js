(static) statistic(u1, s1, n1, u2, s2, n2) → {Array}

Return a test statistic of F-test which follow Snedecor's F distribution. The variance of the sample1 must be larger or equal to sample2.
Parameters:
Name Type Description
u1 Number A mean value (X1) of sample 1 (dummy, not used).
s1 Number A variance (s12) of sample 1.
n1 Number The number of samples (N1) of sample 1.
u2 Number A mean value (X2) of sample 2 (dummy, not used).
s2 Number A variance (s22) of sample 2.
n2 Number The number of samples (N2) of sample 2.

Throws:
Will throw an error when s1 is smaller than s2
Type
Error
Returns:
An array which contains a F-test statistic and a degree of freedom of sample 1 and 2 respectively. [F, ν1, ν2]
Type
Array
Example
// return [2, 2, 1]
statistic.F.statistic(null, 4, 3, null, 2, 2);
// return [3, 3, 3]
statistic.F.statistic(null, 6, 4, null, 2, 3);
// throw an Error('s1 requires to be larger than s2')
statistic.F.statistic(null, 6, 4, null, 12, 3);
// Calculate p-value of two sample distribution
statistic.F.pvalue.apply(null, statistic.F.statistic(null, 6, 4, null, 2, 3));
