range(start, stopopt, stepopt, digitsopt) → {Number}

Create an array with continueous numbers. If the step argument is omitted, it defaults to 1. If the stop argument is omitted, the value of stop is set to start and the value of start become 0. If the step argument is 0, it raise Error. For a positive step, the contents of a range r are determined by the formula r[i] = start + step * i where i >= 0 and r[i] < stop. For a negative step, the contents of the range are still determined by the formula r[i] = start + step * i, but the constraints are i >= 0 and r[i] > stop. If digits argument is specified, it call toFixed method with the value before adding to the contents.
Parameters:
Name Type Attributes Description
start Number A start or stop value.
stop Number <optional>
 A stop value
step Number <optional>
 A step value. Default is 1.
digits Number <optional>
 A digits argument of toFixed method if specified
Source:
  • statistics.js, line 101

Example
